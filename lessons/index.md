All lessons are listed below in order.  Each one builds on the previous, but
most can also be read independently once you have Python installed (Lesson 1).

| # | Title | Learning Objective |
|---|---|---|
| [Lesson 1](01-installation) | Installation & Hello World | By the end of this lesson you will have Python and VS Code installed on your computer (Mac or Windows), understand what a terminal is, and be able to run your first Python script. |
| [Lesson 2](02-functions) | Functions | By the end of this lesson you will be able to assign values to variables, identify Python's four core data types (`int`, `float`, `str`, `bool`), define and call your own functions, understand how parameters and return values work, and explain why breaking code into functions improves readability and reuse. |
| [Lesson 3](03-mathematics) | Mathematical Operations | By the end of this lesson you will be able to use Python's arithmetic operators, understand the difference between `int` and `float`, write functions for Pythagoras' theorem, and be aware of common numeric type gotchas such as floating-point precision. |
| [Lesson 4](04-docstrings) | Docstrings and Code Documentation | By the end of this lesson you will be able to write clear docstrings for your own functions, use `help()` to read documentation, and understand how libraries document themselves the same way. |
| [Lesson 5](05-math-library) | Reading Library Documentation (`math`) | By the end of this lesson you will be able to navigate `docs.python.org`, read a function reference entry, use constants, trigonometric functions, and logarithms from the `math` module, and look up documentation offline with `help()`. |
| [Lesson 6](06-strings) | Strings | By the end of this lesson you will be able to manipulate strings using slicing, formatting, and common methods such as `.split()`, `.join()`, and `.strip()`, and understand how f-strings simplify output. |
| [Lesson 7](07-text-files) | Text Files | By the end of this lesson you will be able to read and write plain-text files using the `with` statement, explain what the `\n` newline character is, and understand how context managers work. |
| [Lesson 8](08-lists) | Lists | By the end of this lesson you will be able to create and manipulate lists, use list methods and slicing, and apply list comprehensions to transform data concisely. |
| [Lesson 9](09-decisions) | Decisions | By the end of this lesson you will be able to use `if`, `elif`, and `else` to control program flow, work confidently with boolean data (`True`/`False`) and boolean operations (`and`, `or`, `not`), and write functions that make decisions. |
| [Lesson 10](10-for-loops) | For Loops with Range | By the end of this lesson you will be able to iterate over sequences and ranges with `for` loops, use `enumerate` and `zip`, and combine loops with lists to process collections of data. |
| [Lesson 11](11-dictionaries) | Dictionaries | By the end of this lesson you will be able to create and use dictionaries to store and look up key–value data, understand what mutable and immutable types are, and iterate over dictionary keys and values. |
| [Lesson 12](12-csv-files) | CSV Files | By the end of this lesson you will understand the CSV format and why it is a universal data exchange standard, be able to read and write CSV files using Python's built-in `csv` module, and write a function that applies a letter-grade mark scheme to a class dataset. |
| [Lesson 13](13-excel-files) | Excel Files | By the end of this lesson you will understand the difference between standard and third-party libraries, be able to read and write `.xlsx` files with `openpyxl`, and convert a CSV file to an Excel workbook. |
| [Lesson 14](14-markdown) | Markdown | By the end of this lesson you will be able to write well-structured Markdown documents with headings, lists, tables, and code blocks, and understand how Jekyll renders Markdown as a website. |
| [Lesson 15](15-git) | Version Control with Git | By the end of this lesson you will be able to initialise a Git repository, make commits, and use `git log` to review history, understanding the three-area model of working directory, staging area, and repository. |
| [Lesson 16](16-github) | GitHub — Remote Repositories | By the end of this lesson you will be able to push a local Git repository to GitHub, pull changes from a remote, and collaborate using the push/pull workflow. |
| [Lesson 17](17-github-pages) | GitHub Pages — Classroom Blog | By the end of this lesson you will have published a live Jekyll classroom blog on GitHub Pages with at least one Markdown post, understanding how static site generation turns Markdown into HTML. |
| [Lesson 18](18-json) | JSON | By the end of this lesson you will understand the JSON format, be able to convert between JSON and Python objects using `json.dumps` and `json.loads`, and build a mark-sheet export to JSON. |
| [Lesson 19](19-uv) | Python Packaging with uv | By the end of this lesson you will be able to create isolated project environments with `uv`, add and remove packages, understand `pyproject.toml`, share your project with `uv sync`, and read a `requirements.txt` file from an older project. |
| [Lesson 20](20-code-reuse) | Code Reuse with Modules | By the end of this lesson you will be able to import built-in modules, write your own reusable module files, explain why modules improve maintainability, and use `if __name__ == "__main__":` as an explicit script entry point. |
| [Lesson 21](21-unit-testing) | Unit Testing | By the end of this lesson you will be able to write and run tests with `pytest`, use `pytest.mark.parametrize` to test many inputs concisely, test that functions raise exceptions correctly, and understand why automated tests are important. |
| [Lesson 22](22-try-except) | Error Handling — try / except | By the end of this lesson you will be able to use `try`, `except`, `else`, `finally`, and `raise` to handle exceptions gracefully, understand which exception types to catch for common errors, and apply error handling to file I/O. |
| [Lesson 23](23-jupyter) | Jupyter Notebooks | By the end of this lesson you will be able to create and run a Jupyter notebook, mix code cells and Markdown cells, and understand when interactive notebooks are more useful than plain Python scripts. |
| [Lesson 24](24-requests) | Requests | By the end of this lesson you will be able to make HTTP GET requests with the `requests` library, parse JSON responses, pass query parameters, and handle HTTP error codes. |
| [Lesson 25](25-requests-retry) | Robust Requests — Retry Mechanics | By the end of this lesson you will be able to write a recursive retry function with exponential back-off, distinguish transient from permanent HTTP errors, and understand how default parameters and recursion work together. |
| [Lesson 26](26-pandas) | Pandas | By the end of this lesson you will be able to load a CSV into a pandas DataFrame, compute descriptive statistics, filter rows, group data, sort, add derived columns, and save results — all with a clear understanding of what each step does and why. |
| [Lesson 27](27-pandas-graphs) | Pandas Graphs | By the end of this lesson you will be able to create bar charts, histograms, scatter plots, box plots, and pie charts from a pandas DataFrame using `matplotlib`, and choose the right chart type for a given question about student data. |
